# Tutorial-Only Blocks

## About Tutorial-Only Blocks

The blocks you see here are custom blocks designed specifically for this tutorial. These blocks help simplify the tutorial steps and make it easier to build your game as you learn.

## What do they do?

These tutorial-specific blocks are designed to:

- Simplify complex game mechanics into easy-to-use blocks
- Keep your code organized and easy to understand
- Help you focus on learning the core concepts
- Make tutorial steps clearer and more straightforward

## How to use them

Just follow the tutorial instructions! As you progress through the tutorial, you'll learn:

1. **What each block does** - The tutorial will explain each block's purpose
2. **When to use them** - You'll see practical examples in context
3. **How they work together** - Build your game step-by-step

## Continue with the tutorial

These blocks are only available within this tutorial. To learn more about what each block does and how to use them, keep following the tutorial steps!
