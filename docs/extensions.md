## Create you own extensions #custom

The [Build Your Own Extension](https://makecode.com/extensions/getting-started) manual is for advanced users who want to publish their own extension.

### ~tip

#### Asset packs

You can create an extension project to provide only assets. This type of project is called an _asset pack_ extension. When an extension is set as an asset pack, only its assets are loaded and any code included is ignored.

Read about [asset packs](https://makecode.com/extensions/asset-packs) to learn more.
